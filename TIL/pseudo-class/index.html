    1. 의사 클래스(pseudo-classes)

    선택하고자 하는 HTML 요소의 특별한 상태(state)를 명시할 때 사용합니다.

    2. 의사 클래스 종류 및 예제

    지금까지 제가 개발하면 자주 사용하던거 위주로 정리하였습니다.

    1. 동적 의사 클래스(dynamic pseudo-classes)

    • :hover : 마우스 호버 시(마우스로 가리킬 때) -:active : 마우스 버튼을 누를 때
      • 클릭하는 순간부터 떼는 순간까지
    • :focus : 사용자가 요소를 클릭 또는 탭하거나 키보드 키로 요소를 선택할 때
      • 마우스가 해당 요소를 클릭해서 포커스가 아웃될 때까지

    2. 상태 의사 클래스(UI element states pseudo-classes)

    • :enabled: 입력, 클릭 등을 활성화거나 포커스를 수락할 수 있는 요소
    • :disabled : enabled과 반대 개념

    3. 구조 의사 클래스(structural pseudo-classes)

    even: 짝수 / odd: 홀수
    An+B → A : 정수 단계 크기 / B : 정수 오프셋 / n : 0부터 시작하는 음이 아닌 정수

    • :first-child : 형제 요소 중에 첫번째 요소

    • :nth-child: 형제 그룹 사이의 위치에 따라 요소를 일치

    • :first-of-type : 형제 요소 중에 자신의 유형과 일치하는 첫번째 요소

    • :nth-of-type : 동일한 유형의 형제 사이의 위치를 ​​기준으로 일치하는 요소
